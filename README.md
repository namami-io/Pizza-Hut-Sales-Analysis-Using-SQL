# Pizza-Hut-Sales-Analysis-Using-SQL

This project demonstrates SQL skills by analyzing Pizza Hut sales data. The analysis helps answer key business questions related to revenue, sales performance, and customer preferences.

# Tools
- MySQL
- MySQL Workbench

#  Skills
Joins • Subqueries • Window Functions • GROUP BY • ORDER BY • WHERE • LIMIT • Aggregate Functions

# 📈 Insights
- Total Revenue
- Cumulative Revenue
- Highest Selling Pizzas
- Top 3 Pizza Types
- Order Trends
- Category-wise Sales
